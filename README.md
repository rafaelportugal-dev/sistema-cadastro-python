# Sistema de Cadastro em Python

Sistema de cadastro de usuários desenvolvido em Python com interface em terminal, validação de dados e armazenamento em arquivo JSON.

## Funcionalidades
- Cadastro de usuários
- Listagem de usuários cadastrados
- Busca de usuários por nome
- Atualização de dados
- Remoção de usuários
- Validação de nome, idade e email
- Salvamento automático em arquivo JSON

## Tecnologias
- Python
- JSON

## Objetivo
Projeto desenvolvido para praticar lógica de programação, estruturas de dados, validação de entrada e persistência de dados.

## Como executar
1. Instale o Python
2. Baixe os arquivos do projeto
3. Execute o comando:

python main.py
